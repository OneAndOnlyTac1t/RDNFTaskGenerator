		
		ФУНКЦІОНАЛ
Дана програма дозволяє генерувати завдання для тестової системи Moodle, автоматично генерувати ДДНФ булевих функцій для функцій визначеними від 2 до 4 змінних, проводити алгоритм спрощення, редагувати вміст питання.

		Принципи використання програми.
Кнопка "Визначити" відкриває діалогове вікно, яке дозволяє задати спільне для всіх питань повідомлення. ВАЖЛИВО!!! Існуючі питання зі списку питань вже мають своє повідомлення, задавати повідомлення необхідно перед тим, як добавляти питання в список.

Кнопка "Зберегти" відкриває діалогове вікно для збереження файлу. Збережені будуть всі питання, які додані до списку запитань у форматі Moodle XML.

Комбобокс "Кількість змінних" необхідний для автоматичного генерування ДДНФ. В цьому комбобоксі встановлюється кількість змінних, для булевої функції, ДДНФ якої буде згенеровано.

Комбобокс "Кількість кон'юнкцій" необхідний для автоматичного генерування ДДНФ.В цьому комбобоксі встановлюється кількість елементарних кон'юнкцій в автоматично згенерованій ДДНФ.

Кнопка "Згенерувати" необхідна для процесу автоматичного генерування ДДНФ за визначеними параметрами з комбобоксів. Згенерована ДДНФ з'явиться у полі "ДДНФ"

Поле "ДДНФ" необхідне для ручного введення, редагування існуючої ДДНФ, для відображення згенерованої автоматично ДДНФ.
	Правила коректного вводу ДДНФ для подальшого правильного парсингу:
	1) кожен співмножник повинен бути відокремлений пробілом з обох сторін
	2) знак диз'юнкції (V) повинен бути відокремлений пробілом з обох сторін
	3) для заперечення слід використовувати символ "!"
	4) пробілів на початку та в кінці запису бути не повинно
	5) використовувати можна будь які літери кирилиці або латиниці, окрім літери "V", адже вона використовується як знак диз'юнкції.

Кнопка "Обрахувати" необхідна для виконання алгоритму спрощення введеної в поле "ДДНФ" ДДНФ булевої функції. Результат виконання алгоритму відображений в полі "Результат", яке доступне тільки в режимі читання.

Кнопка "Додати" необхідна для того, щоб дадати пару ДДНФ та обчисленої ТДНФ до списку запитань. До кожного запитання додається останнє повідомлення, яке визначене за допомогою кнопки "Визначити".

Кнопка "Видалити" необхідна для того, щоб видалити виділене питання зі списку питань. Для того, щоб виділити запитання необхідно натиснути на нього ліву кнопку миші у ЛістВью з запитаннями.

Кнопка "Показати запитання" відкриває діалогове вікно, яке дозволяє показати сформоване запитання зі списку питань у вигляді, в якому воно буде представлене у системі Moodle під час виконнання тестового завдання. Відповідь до сформованого запитання записана у окремому полі. !!!ОБИДВА ПОЛЯ НЕ ПІДЛЯГАЮТЬ РЕДАГУВАННЮ!!!

ЛістВью, який розташований знизу показує список сформованих питань у вигляді ДДНФ та спрощеної за алгоритмом ТДНФ.


		Приклад задання такого коментаря до запитання, який би міг стати поясненням для студента:

Використовуючи алгоритм спрощення спростити задану ДДНФ. Правила правильного вводу результату 1)кожен множник повинен бути відокремлений пробілом від іншого співмножника 2)знак диз'юнкції (V) повинен бути відокремлений пробілом з обох боків 3)пробілів на початку та вкінці відповіді бути не повинно. 4)Порядок елементарних кон'юнкцій та співмножників повинен бути строго відповідним порядку при виконанні алгоритму.
